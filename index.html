simple web app

Map (Google) — Route + Live Tracking

Add your Google Maps API key in the script URL, then use GPS or import a .gpx route.

Cost guard (keep it free)
  • Lazy init: the map does not load until you click a button (so no tile load if you never use it).
  • One map instance: we reuse a single map and avoid reloading the API.
  • Only Maps JS + geometry: no Places/Routes/paid geolocation calls.
  • Key restricted: lock your key to your Pages domain and the Maps JavaScript API only.
  • Quotas & budgets: set tiny daily caps and a $0 budget alert in Google Cloud.
